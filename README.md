## Calculadora IMC

Projeto simples em Python que calcula o Índice de Massa Corporal (IMC) com base em seu peso e altura.

## Funcionalidades
- O usúario informa seu nome, peso e altura
- Calcula o IMC
- Exibe resultado e classificação (abaixo do peso, normal, sobrepeso, etc.)

## Como rodar
1 - Abra o terminal na pasta do projeto
2- Excute:

python calculadora_imc.py
